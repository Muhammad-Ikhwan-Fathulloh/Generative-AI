Evolusi Komunikasi: Prompt

Apa itu Prompt?

Input tekstual yang dikirimkan ke model AI (seperti GPT-4, Gemini, atau Claude) untuk memandu tugas tertentu.

"Prompt bertindak sebagai pemicu (trigger) bagi probabilitas statistik model untuk menghasilkan urutan kata tertentu."

Prompt Engineering

Disiplin ilmu untuk mengoptimalkan input guna mendapatkan respons terbaik dari AI.

  • Iterasi berkelanjutan (Trial & Error).
  • Penggunaan teknik *Zero-shot* hingga *Few-shot*.
  • Penyelarasan antara niat manusia dan bahasa mesin.
Sumber: DAIR.AI Prompt Engineering Guide & OpenAI Documentation
KOALAWAN

Anatomi Sebuah Prompt

Prompt yang efektif biasanya mengandung kombinasi dari elemen berikut:

1. Instruksi

Tugas Spesifik: Apa yang harus dilakukan AI? (Contoh: "Ringkas", "Tulis", "Terjemahkan").

2. Konteks

Latar Belakang: Siapa audiensnya? Apa gaya bahasanya? (Contoh: "Untuk anak usia 5 tahun").

3. Data Input

Bahan Olahan: Data mentah yang perlu diproses. (Contoh: Teks artikel, sekumpulan angka).

4. Indikator Output

Format Hasil: Bagaimana bentuk akhirnya? (Contoh: "Tabel", "JSON", "Markdown").

"Klasifikasikan [Instruksi] sentimen dari ulasan berikut [Konteks]: 'Makanannya enak tapi pelayanannya lama' [Input]. Berikan hasil dalam format JSON [Output]."

Contoh Kasus: Optimasi Nama Produk

Bad Prompt (Ambigu)

"Berikan saya ide nama untuk produk pakaian olahraga."

Hasil AI: Sporty, Fitwear, ActiveStyle (Terlalu umum & membosankan).

Good Prompt (Engineerized)

"Saya sedang membangun brand pakaian olahraga ukuran besar (plus-size) yang premium. Berikan 5 ide nama unik yang mengandung kata 'Fellas' atau 'Big-Bro'. Gunakan nada bicara yang keren dan inklusif."

Hasil AI: Big-Bro Elite, Fellas Flex, Grande Bro, Fellas Prime, Big-Bro Aura.

Prinsip 1 & 2: Direction & Format

1. Give Direction

Berikan "Persona" atau gaya bicara khusus.

"Tulis penjelasan tentang AI seolah-olah kamu adalah Steve Jobs yang sedang melakukan keynote."

Hasil akan menggunakan kata-kata seperti: "Revolutionary", "Magical", "One more thing".

2. Specify Format

Mengarahkan struktur data agar siap pakai.

Format Kegunaan
Table Perbandingan data
JSON Integrasi aplikasi
CSV Import ke Excel
KOALAWAN

Prinsip 3 & 4: Few-Shot & Iteration

3. Provide Examples (Few-Shot)

AI belajar dari pola. Berikan 2-3 contoh input/output sebelumnya.

"Ubah nama produk ke gaya futuristik.
Sepatu -> AeroFoot 2099
Jam -> ChronoLink X
Tas -> ..."

Teknik ini meningkatkan akurasi hingga 40%.

4. Evaluate Quality

Jangan berhenti di respons pertama. Berikan umpan balik.

User: "Terlalu formal, coba buat lebih santai."

User: "Ganti semua kata 'Anda' menjadi 'Kamu'."

User: "Ulangi poin ke-3, buat lebih detail."
