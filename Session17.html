Memahami Prompt Injection

Apa itu Prompt Injection?

Upaya untuk memanipulasi instruksi sistem dengan menyisipkan perintah baru (Jailbreak) untuk melewati hambatan keamanan AI.

  • Direct Injection: User memberikan perintah nakal.
  • Indirect Injection: Melalui dokumen atau link yang dibaca AI.
  • Data Poisoning: Memberikan data bias untuk merusak logika AI.

Cara Pencegahan:

  • Gunakan System Instructions yang sangat kuat.
  • Validasi Input sebelum dikirim ke LLM.
  • Gunakan Separation of Concerns (Pemisahan Instruksi & Data).
